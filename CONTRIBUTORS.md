## 🎯 How to Become a Contributor

### Contribution Levels

| Level | Requirements | Benefits |
|-------|-------------|----------|
| 🥉 **Bronze** | 1-2 PRs | • GitHub contributors list<br/>• Discord contributor role |
| 🥈 **Silver** | 3-5 PRs | • README contributor wall<br/>• Discord special role<br/>• Swag sticker |
| 🥇 **Gold** | 6+ PRs | • Core member status<br/>• Write access<br/>• Decision making<br/>• Swag pack |

### Quick Ways to Contribute

| Task | Time | Difficulty |
|------|------|------------|
| 🐛 Fix a typo | 5 min | ⭐ Easy |
| 🔗 Fix broken link | 10 min | ⭐ Easy |
| 📝 Add a resource | 15 min | ⭐⭐ Medium |
| 💡 Improve description | 10 min | ⭐ Easy |
| 📖 Write a tutorial | 1-2 hrs | ⭐⭐⭐ Hard |
| 🛠️ Add a skill | 2-4 hrs | ⭐⭐⭐⭐ Expert |
