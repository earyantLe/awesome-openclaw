## 🚀 快速开始 / Quick Start

### 5 分钟快速上手 / 5-Minute Quick Start

```bash
# 1. 安装 OpenClaw / Install OpenClaw
npm install -g openclaw

# 2. 初始化项目 / Initialize project
openclaw init my-first-agent

# 3. 启动 Gateway / Start Gateway
cd my-first-agent
sh scripts/start.sh
```

**完成！/ Done!** 🎉

现在你可以开始学习了！/ Now you can start learning!

## 💡 学习技巧 / Learning Tips

### ✅ 应该做的 / Do's

1. **循序渐进** - 按照学习路径逐步学习
   **Follow the path** - Learn step by step

2. **动手实践** - 每个教程都要实际运行代码
   **Hands-on practice** - Run the code for each tutorial

3. **做笔记** - 记录关键知识点和遇到的问题
   **Take notes** - Record key points and issues encountered

4. **参与社区** - 在 Discord 提问和分享
   **Join community** - Ask and share in Discord

### ❌ 不应该做的 / Don'ts

1. **不要跳跃学习** - 基础很重要
   **Don't skip** - Basics are important

2. **不要只看不练** - 实践出真知
   **Don't just watch** - Practice makes perfect

3. **不要害怕犯错** - 错误是最好的老师
   **Don't fear mistakes** - Mistakes are the best teachers
