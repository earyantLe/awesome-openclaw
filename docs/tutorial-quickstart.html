Quick Start ⏱️ 5 minutes 📊 Beginner

What You'll Learn

  • Install OpenClaw
  • Initialize your workspace
  • Start the gateway
  • Verify installation

Step 1: Install OpenClaw

npm install -g openclaw

Step 2: Initialize Workspace

openclaw init my-agent
cd my-agent

Step 3: Start Gateway

openclaw gateway start

Step 4: Verify

openclaw status
