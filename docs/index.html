🦞 What is OpenClaw?

OpenClaw is an open-source AI agent runtime that transforms your LLM into a proactive, tool-using assistant with memory, scheduling, and real-world integrations.

🧠

Memory-Equipped

Persistent memory files for long-term context

⏰

🔌

Plugin Architecture
