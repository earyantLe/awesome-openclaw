## 🦞 什么是 OpenClaw？

[OpenClaw](https://github.com/openclaw/openclaw) 是一个开源 AI 助手运行时，可将你的 LLM 转变为**主动、使用工具的助手**，具有记忆、调度和真实世界集成能力。

### 为什么选择 OpenClaw？

| 特性 | 描述 |
|------|------|
| 🧠 **记忆系统** | 持久化记忆文件，支持长期上下文 |
| ⏰ **定时调度** | 内置任务调度器，支持主动工作 |
| 🛠️ **工具生态** | 可扩展技能系统，50+ 集成 |
| 💬 **多通道支持** | Discord、Telegram、WhatsApp、飞书、Slack、Signal |
| 🔌 **插件架构** | 轻松扩展自定义集成 |

## 🤝 贡献

欢迎贡献！以下是参与方式：

1. **Fork** 仓库
2. **创建** 分支 (`git checkout -b feature/add-my-skill`)
3. **添加** 你的资源，并附上清晰描述
4. **提交** 更改 (`git commit -m 'Add: My awesome skill'`)
5. **推送** 到分支 (`git push origin feature/add-my-skill`)
6. **提交** Pull Request

### 贡献指南

- ✅ 资源应与 OpenClaw 生态相关
- ✅ 描述清晰简洁（1-2 句）
- ✅ 提交前测试所有链接
- ✅ 在分类中按字母顺序排列
- ✅ 使用正确的 markdown 格式
